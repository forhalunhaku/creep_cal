# fib Model Code 2010 徐变模型说明

本文档与当前应用的计算逻辑一致。MC2010 输出徐变系数 `φ(t,t0)`，并将其分为基本徐变和干燥徐变两部分。

## 输入参数

| 界面含义 | 批量列名 | 单位 | 说明 |
| --- | --- | --- | --- |
| 平均抗压强度 | `fcm` | MPa | 混凝土平均抗压强度 |
| 相对湿度 | `RH` | % | 环境相对湿度 |
| 加载龄期 | `t0` | d | 混凝土开始受荷的龄期 |
| 截面面积 | `Ac` | mm2 | 构件截面面积 |
| 暴露周长 | `u` | mm | 与环境接触的截面周长 |
| 温度 | `T` | C | 养护或环境温度 |
| 水泥等级 | `Cs` | - | 可选：`32.5N`、`32.5R`、`42.5N`、`42.5R`、`52.5N`、`52.5R` |
| 计算时间 | `t` | d | 从浇筑起算的目标龄期 |

当 `t <= t0` 时，应用返回 `φ = 0`。

## 水泥等级系数

| `Cs` | `a` |
| --- | ---: |
| `32.5N` | -1 |
| `32.5R` | 0 |
| `42.5N` | 0 |
| `42.5R` | 1 |
| `52.5N` | 1 |
| `52.5R` | 1 |

未识别的水泥等级按 `a = 0` 处理。

## 计算公式

温度修正龄期：

```math
t_{0T}=t_0e^{13.65-\frac{4000}{273+T}}
```

水泥等级修正后的加载龄期：

```math
t_{0,adj}=t_{0T}\left(\frac{9}{2+t_{0T}^{1.2}}+1\right)^a
```

截面特征长度：

```math
h=\frac{2A_c}{u}
```

强度与尺寸参数：

```math
\alpha_f=\sqrt{\frac{35}{f_{cm}}}
```

```math
\beta_h=\min(1.5h+250\alpha_f,\ 1500\alpha_f)
```

时间差：

```math
\Delta t=t-t_0
```

基本徐变：

```math
\phi_{bc}
=\frac{1.8}{f_{cm}^{0.7}}
\ln\left[\left(\frac{30}{t_{0,adj}}+0.035\right)^2\Delta t+1\right]
```

干燥徐变指数：

```math
\gamma(t_0)=\frac{1}{2.3+\frac{3.5}{\sqrt{t_{0,adj}}}}
```

干燥徐变：

```math
\phi_{dc}
=\frac{412}{f_{cm}^{1.4}}
\frac{1-\frac{RH}{100}}{\left(0.1\frac{h}{100}\right)^{1/3}}
\frac{1}{0.1+t_{0,adj}^{0.2}}
\left(\frac{\Delta t}{\beta_h+\Delta t}\right)^{\gamma(t_0)}
```

总徐变系数：

```math
\phi(t,t_0)=\phi_{bc}+\phi_{dc}
```

## 输出

| 输出 | 单位 | 说明 |
| --- | --- | --- |
| `φ(t,t0)` | - | 徐变系数 |

单点计算页面会生成 0 到 10000 天的曲线；批量计算按导入表格中的 `t` 输出对应时间点的结果。

## 使用说明

- `RH` 按百分数输入，例如 70 表示 70%。
- `Ac` 与 `u` 必须使用一致的长度单位；当前实现按 `mm2` 和 `mm` 解释。
- 当前实现不对输入范围做规范适用性裁剪，超出常规工程范围时结果仅代表公式外推。
