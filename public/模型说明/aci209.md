# ACI 209R-92 徐变模型说明

本文档与当前应用的计算逻辑一致。ACI 209 输出徐变系数 `φ(t,t0)`，用于描述加载后混凝土徐变随时间的发展。

## 输入参数

| 界面含义 | 批量列名 | 单位 | 说明 |
| --- | --- | --- | --- |
| 加载龄期 | `t0` | d | 混凝土开始受荷的龄期 |
| 相对湿度 | `H` | % | 环境相对湿度 |
| 体积表面积比 | `VS` | mm | 构件体积与暴露表面积之比 |
| 砂率 | `sphi` | - | 参与 `βsφ` 计算的砂率 |
| 水泥用量 | `Cc` | kg/m3 | 单位体积混凝土水泥用量 |
| 含气量 | `alpha` | - | 参与 `βα` 计算的含气量 |
| 计算时间 | `t` | d | 从浇筑起算的目标龄期 |

当 `t <= t0` 时，应用返回 `φ = 0`。

## 计算公式

记：

```math
\Delta t = t - t_0
```

徐变系数：

```math
\phi(t,t_0)=\beta_c(\Delta t)\phi_\infty
```

终极徐变系数：

```math
\phi_\infty
=2.35\beta_{t0}\beta_{RH}\beta_{VS}\beta_{s\phi}\beta_{Cc}\beta_\alpha
```

各修正项为：

```math
\beta_{t0}=1.25t_0^{-0.118}
```

```math
\beta_{RH}=1.27-0.0067H
```

```math
\beta_{VS}=\frac{2(1+1.13e^{-0.0213VS})}{3}
```

```math
\beta_{s\phi}=0.88+0.244s_\phi
```

```math
\beta_{Cc}=0.75+0.00061C_c
```

```math
\beta_\alpha=0.46+9\alpha
```

时间发展项：

```math
\beta_c(\Delta t)=\frac{\Delta t^{0.6}}{10+\Delta t^{0.6}}
```

## 输出

| 输出 | 单位 | 说明 |
| --- | --- | --- |
| `φ(t,t0)` | - | 徐变系数 |

单点计算页面会生成 0 到 10000 天的曲线；批量计算按导入表格中的 `t` 输出对应时间点的结果。

## 使用说明

- `H` 按百分数输入，例如 70 表示 70%。
- `sphi` 表示砂率，`alpha` 表示含气量，二者按无量纲小数输入。
- 当前实现不对输入范围做规范适用性裁剪，超出常规工程范围时结果仅代表公式外推。
